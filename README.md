# GearRent Pro - Equipment Rental Management System

## Project Overview
This is a JavaFX-based management system designed for professional equipment rentals. It features role-based access control, branch management, and automated rental calculations.

## Getting Started
1. **Database Setup**: Import the `Dump20251226 (1).sql` file into your MySQL server to recreate the schema and sample data.
2. **Configuration**: Ensure your database connection settings in the source code match your local MySQL credentials.
3. **Run**: Execute the `AppInitializer.java` file to launch the application.
